# Sprite Studio v1.0.5

Release date: 2026-04-02

## Highlights

- Stabilized the embedded photo editor after several regressions around layer creation, crop selection, and toolbar behavior.
- Added persistent application logs with automatic cleanup for easier troubleshooting.
- Improved local install behavior with safer default paths and clearer AI setup notes.

## Photo editor fixes

- Crop mode no longer creates a default selection rectangle before the first drag.
- Direct graphic-layer creation now avoids duplicate popup-driven layer creation.
- Added a mirrored color swatch button to the top-right editor header while keeping the original palette control in place.
- Reduced regressions from toolbar customization experiments by keeping the original 1-column toolbox flow intact.

## Diagnostics

- Added rolling log files under the app user-data folder.
- Renderer load failures, renderer console errors, worker failures, and uncaught process errors are now written to disk.
- Old logs are cleaned up automatically by age, count, and total size.

## Packaging and install

- Application version bumped to `1.0.5`.
- Installer and portable package can both be produced from the repository root.
- Default project storage remains inside the app user-data tree instead of `Documents`.

## Notes

- AI tools still require internet access plus local `Git for Windows` and `Node.js 20+` on first setup.
- `WORKFLOW.md` was not included in this release commit because it was already deleted in the local worktree outside the scoped release changes.
