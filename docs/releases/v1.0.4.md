# Sprite Studio v1.0.4

Release date: 2026-04-02

## Highlights

- Improved timeline workflow with better selection handling and quicker bulk actions.
- Reworked inspector, left panel, pixel helper, and Leshy Sprite flows for faster day-to-day editing.
- Added startup option to launch minimized.
- Reduced renderer startup risk by hardening error handling and removing a problematic runtime import.
- Removed Unity `.anim` guidance from the app-facing workflow.

## UI and workflow improvements

- Timeline now supports stronger multi-selection flow, bulk delay editing, and richer status feedback.
- Left panel now exposes better browsing helpers for clip-heavy projects.
- Inspector is reorganized into more readable grouped sections.
- Background removal and IOPaint flows show clearer processing and retry states.
- Pixel helper and Leshy Sprite tools now include faster preset-based workflows.

## Stability fixes

- Fixed a renderer blank-screen issue caused by an unsafe runtime import path in the inspector.
- Added renderer crash and load-failure diagnostics in the Electron shell.
- Changed the photo editor crop tool so no crop rectangle appears until the user starts dragging.
- Adjusted the embedded photo editor toolbar toggle to cycle through 1-column, 2-column, and hidden states.
- Reduced custom layer-add interception in the embedded photo editor to avoid layer stack corruption.

## Packaging

- Application version bumped to `1.0.4`.
- Added dedicated app icon assets for Windows packaging.

## Notes

- `Sprite Studio.exe` in the repository root is treated as a local build artifact and is not part of the source release commit.
