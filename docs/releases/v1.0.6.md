# Sprite Studio v1.0.6

Release date: 2026-04-02

## Highlights

- Updated the pixel editor working branch on top of the latest `v1.0.5` codebase without carrying older unrelated local changes.
- Improved the built-in AI inpainting flow with clearer mode naming and a visible brush-size cursor in manual masking mode.
- Refined the embedded photo editor header by keeping only the mirrored color control and removing the redundant reload button.

## AI editing

- Renamed the built-in inpainting tabs so the simpler hosted workflow is now labeled `빠른 편집`.
- Renamed the local mask-painting workflow to `매뉴얼 편집` to better match how it behaves.
- Added a live circular brush cursor to manual inpainting so the on-canvas cursor reflects the current brush size instead of showing only a crosshair.

## Photo editor

- Fixed the top-right mirrored color swatch so it now reflects the currently selected color.
- Removed the extra reload button from the photo editor header.

## Packaging and install

- Application version bumped to `1.0.6`.
- Installer and portable package names now use `1.0.6`.
